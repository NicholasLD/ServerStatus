A [Flarum](http://flarum.org) extension. serverstatus

## Installation

Install with composer:

```sh
composer require nicholasld/serverstatus:"*"
```

## Updating

```sh
composer update nicholasld/serverstatus:"*"
php flarum migrate
php flarum cache:clear
```
